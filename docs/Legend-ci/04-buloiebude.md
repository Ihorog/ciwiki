# Було → Є → Буде

> [← Повернутись до Legend Ci](README.md)

## Центральна концепція часових станів

**Було → Є → Буде** — це ключова темпоральна логіка системи Cimeika, яка організує всі процеси, події та знання вздовж часової осі.

## Три стани

### Було (минуле)

**Призначення**: контекст, пам'ять, архів

**Характеристики**:
- Зафіксовані події та рішення
- Історія розвитку ідей
- Досвід і вивчені уроки
- База для аналізу та рефлексії

**Ключові слова**: пам'ять, архів, історія, контекст

### Є (теперішнє)

**Призначення**: дія, фокус, потік

**Характеристики**:
- Поточний стан системи
- Активні процеси
- Точка прийняття рішень
- Момент «тут і зараз»

**Ключові слова**: фокус, потік, дія, присутність

### Буде (майбутнє)

**Призначення**: прогноз, сценарій, план

**Характеристики**:
- Можливі траєкторії розвитку
- Плановані дії та події
- Передбачення результатів
- Стратегічне бачення

**Ключові слова**: сценарій, план, передбачення, потенціал

## Застосування в модулях

### ПоДія

Кожна подія має три фази:
- **Було**: що передувало події
- **Є**: що відбувається зараз
- **Буде**: що буде після події

### Казкар

Наративна структура будується на трьох часах:
- **Було**: історія, передісторія
- **Є**: кульмінація, дія
- **Буде**: розв'язка, наслідки

### Календар

Часові маркери організовані по станах:
- **Було**: історія подій (архів)
- **Є**: поточний період (фокус)
- **Буде**: майбутні плани (прогноз)

### Ci (Центральний інтелект)

Ci забезпечує єдину логіку переходів між станами:

```
Було → оцінка контексту → Є → генерація сценаріїв → Буде
```

## Резонансні періоди

Стани «було/є/буде» працюють на різних часових масштабах:

| Період | Було | Є | Буде |
|---|---|---|---|
| День | вчора | сьогодні | завтра |
| Тиждень | минулий | поточний | наступний |
| Місяць | попередній | цей | майбутній |
| Квартал | минулі 3 міс. | поточні 3 міс. | наступні 3 міс. |

## Контрольні питання

При роботі з будь-яким модулем системи:

- **Було**: Який контекст? Що передувало?
- **Є**: Що змінюємо зараз? Чому?
- **Буде**: Які наслідки? Що далі?

## Сенсові ваги

Ci присвоює вузлам різні ваги залежно від їх темпоральної значущості:

- **Short-term** (короткострокові): фокус на «є»
- **Long-term** (довгострокові): зв'язок «було → буде»
- **Резонансні** (ключові): впливають на всі три стани одночасно
