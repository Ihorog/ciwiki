# Функції та можливості

> [← Повернутись до Legend Ci](README.md)

## Функції Ci як оркестратора

### Оркестрація модулів

**Ci** координує взаємодію між всіма модулями Cimeika:

- **ПоДія** ініціює події
- **Настрій** модулює емоційний контекст
- **Маля** генерує творчий контент
- **Казкар** створює наративні зв'язки
- **Календар** прив'язує до часової осі
- **Галерея** візуалізує процеси

### Синергія («1+1=3»)

Ci не просто координує модулі — він створює синергетичний ефект:

- Сума взаємодій більша за окремі частини
- Резонансні зв'язки між подіями та образами
- Емерджентні властивості системи

### Подієва шина

Ci використовує подієву архітектуру для зв'язку модулів:

```
ci://events — центральна шина подій
ci://gallery — візуальний потік
ci://calendar — часові маркери
```

## Функції Legend Ci як бази знань

### Збереження знань

Legend Ci зберігає:

- Концептуальні основи Cimeika
- Історію еволюції системи
- Практики та процедури
- Зв'язки між ідеями

### Формат вузлів

Короткі вузли (1–3 речення) із структурою:

- **Час**: коли виникла ідея
- **Сенс**: що це означає
- **Стан**: було/є/буде
- **Зв'язки**: з якими концепціями пов'язано

### Навігація

Система тегів і посилань забезпечує:

- Швидкий пошук за темою
- Перегляд еволюції ідей
- Зв'язки між модулями
- Хронологічну послідовність

## Практичні функції

### Для розробників

- Єдине джерело істини про архітектуру
- Еталон термінології та концепцій
- Історія технічних рішень

### Для користувачів

- Зрозумілі пояснення складних ідей
- Інтерактивні приклади та активності
- Емоційно насичені описи

### Для дослідників

- Систематизація спостережень
- Зв'язки між науковими фактами та практикою
- База для експериментів

## Інтерактивні елементи

Legend Ci підтримує:

- **Експерименти**: прості досліди для перевірки ідей
- **Спостереження**: практики уважності
- **Ігри**: активності для засвоєння концепцій
- **Візуалізації**: схеми, діаграми, ескізи

## Контракти даних

Ci працює через чіткі контракти:

- YAML/JSON схеми для обміну даними
- Версіонування інтерфейсів
- Мінімальне ядро, розширення плагінами
