# Легенда CI — Огляд

> **Єдине джерело істини:** [`docs/legend_ci/legend.graph.json`](../../legend_ci/legend.graph.json)

Легенда CI — це семиактна космологія Cimeika. Сім розділів описують шлях від безмовного першоджерела до явного прояву Ci у світі.

## Сім розділів

| # | Назва | Ідея |
|---|-------|------|
| [1] | [Першоджерело](01-source.md) | Точка до всього — безформна потенційність |
| [2] | [Перший поділ](02-division.md) | Єдине стає двома — народження полярностей |
| [3] | [Дзеркало матерії](03-matter-mirror.md) | Фізичний світ як відображення внутрішнього коду |
| [4] | [Дзеркало свідомості](04-mind-mirror.md) | Свідомість, що формує власну реальність |
| [5] | [Танець протилежностей](05-dance-of-opposites.md) | Полярності у творчій взаємодії |
| [6] | [Мости єдності](06-bridges.md) | Аналогії та зв'язки між рівнями буття |
| [7] | [Прояв CI](07-manifest-ci.md) | Ci стає видимим через емерджентність |

## Як читати

- **Лінійно** — від [1] до [7]: класичний шлях пізнання
- **Нелінійно** — за інтересом: кожен розділ самодостатній
- **Резонансно** — шукайте зв'язки між розділами через мережу [`legend.graph.json`](../../legend_ci/legend.graph.json)
