# Легенда Ci — безкінечна вікі

> Призначення: «еталон знань» про походження, еволюцію та практики Cimeika.

## Правила легенди
- Додаємо короткими вузлами (1–3 речення)
- Кожен вузол має: час, сенс, зв’язки, стан (**було/є/буде**)
- Зв’язуємо з Подіями, Настроєм, Галереєю

## Формат вузла
| Поле | Опис | Приклад |
|---|---|---|
| Час | ISO-мітка | 2025-08-29 |
| Сенс | коротко | «запуск структури 7×» |
| Стан | було/є/буде | є |
| Зв’язки | теги | #Ci #Казкар |

## Стартові вузли
- 2025-08-29 — «Фіксація структури 7 розділів», стан: **є**, зв’язки: #Ci #Cimeika
- 2025-08-29 — «Зав’язка Казкаря як наративного ядра», стан: **є**, зв’язки: #Казкар #Легенда

## Далі
- Щодня: 1 вузол «що відбулося / що буде»
- Щотижня: короткий дайджест із тегів
