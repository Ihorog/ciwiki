# Казкар — Наратив, пам’ять, смисли

> Роль: оповідь як «клей» системи; архетипи, жанри, мотиви.

## Навіщо
- Пояснювати події через історії
- Зберігати «Легенду Ci» як безкінечну вікі

## Унікальні риси
- 7 блоків × 30 ключів; стани **було / є / буде**
- 5 стихій: вогонь, вода, земля, повітря, ефір
