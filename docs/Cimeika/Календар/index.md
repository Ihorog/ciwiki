# Календар — Ритм і прив'язка у часі

> Роль: тримає часові координати, ритми, повторюваність.

## Навіщо
- Підсвічувати «коли саме»; фіксація прогресу
- Пов’язувати події з історіями та настроєм

## Унікальні риси
- Маркери «було/є/буде» на часовій осі
- Резонансні періоди (тиждень/місяць/квартал)

## Характеристики
| Період | Використання | Теги |
|---|---|---|
| День | чек-ін/чек-аут | #щодня |
| Тиждень | огляд, планування | #ритм |
| Місяць | фази, підсумки | #цикл |

## Далі
- Автонумерація «вузлів тижня»
- Експорт оглядів для Казкаря
