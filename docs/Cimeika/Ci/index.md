# Ci — Центральний інтелект

> Роль: ядро оркестрації сенсів, даних і взаємодій у Cimeika.

## Навіщо
- Узгоджує модулі (ПоДія, Настрій, Маля, Казкар, Календар, Галерея)
- Забезпечує єдину логіку станів: **було → є → буде**

## Архітектурні принципи
- Мінімальне ядро, розширення плагінами
- Подієва шина взаємодій
- Контракти даних (YAML/JSON схемі)

## Унікальні риси
- «1+1=3» (синергія модулів)
- Сенсові ваги вузлів (short/long-term)
- Резонансні зв’язки між подіями та образами

## Стани
| Стан | Призначення | Ключові слова |
|---|---|---|
| Було | контекст | пам’ять, архів |
| Є | дія | фокус, потік |
| Буде | прогноз | сценарій, план |

## Інтеграції (макети)
- `ci://events` — шина подій
- `ci://gallery` — візуальний шар
- `ci://calendar` — часові мітки

- Який мінімальний крок з найбільшим впливом?
