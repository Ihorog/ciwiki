# Ci — Центральний інтелект

> **Примітка**: Повна концептуальна документація про Ci тепер знаходиться в [Legend Ci](../../../Legend%20ci/README.md) — єдиному джерелі істини для всіх концепцій Cimeika.

## Коротко

**Ci** (Центральний інтелект) — ядро оркестрації сенсів, даних і взаємодій у Cimeika.

### Ключові функції

- Узгоджує модулі (ПоДія, Настрій, Маля, Казкар, Календар, Галерея)
- Забезпечує єдину логіку станів: **було → є → буде**
- Створює синергію між модулями («1+1=3»)
