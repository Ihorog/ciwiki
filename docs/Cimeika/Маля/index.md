# Маля — Творчість і навчання

> Роль: простір гри, уяви, навчальних активностей.

## Навіщо
- Легкий вхід у творчість
- Матеріал для Галереї, Казкаря

## Унікальні риси
- «Мікрозавдання» 5–10 хв
- Образні теги для сюжетів Казкаря

## Механіка
| Крок | Дія | Вихід |
|---|---|---|
| 1 | обрати тему | промпт |
| 2 | зробити спробу | чернетка |
| 3 | зберегти | слайд/картка |

## Далі
- Шаблони завдань за віком
- Синхрон із Календарем (розклад)
