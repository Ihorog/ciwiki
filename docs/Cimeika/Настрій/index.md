# Настрій — Емоційні стани

> Роль: емоційний мод у взаємодії, вплив на сценарії.

## Навіщо
- Пояснювати якість дій: «як відчувається»
- Мапити емоцію на подію та результат

## Унікальні риси
- Вектори: спокій ↔ напруга, радість ↔ смуток
- Вплив на рішення (гальмує / прискорює)

## Емо-вектори (ядро)
| Вісь | Мінус | Плюс |
|---|---|---|
| Напруга | хаос | порядок |
| Енергія | виснаж | підйом |
| Контакт | ізоляція | зв’язок |

## Далі

- Підказки дій «за станом» (мікро-рекомендації)
